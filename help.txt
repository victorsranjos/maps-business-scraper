Usage: convex <command> [options]

Start developing with Convex by running `npx convex dev`.

Commands:
  dev [options]                        Develop against a dev deployment,
                                       watching for changes
  deploy [options]                     Deploy to your prod deployment
  run [options] <functionName> [args]  Run a function (query, mutation, or
                                       action) on your deployment
  import [options] <path>              Import data from a file to your
                                       deployment
  dashboard|dash [options]             Open the dashboard in the browser
  docs [options]                       Open the docs in the browser
  logs [options]                       Watch logs from your deployment
  export [options]                     Export data from your deployment to a ZIP
                                       file
  env [options]                        Set and view environment variables
  data [options] [table]               List tables and print data from your
                                       database
  codegen [options]                    Generate backend type definitions
  update                               Print instructions for updating the
                                       convex package
  logout                               Log out of Convex on this machine
  function-spec [options]              List function metadata from your
                                       deployment
  insights [options]                   Show health insights for your deployment
  disable-local-deployments [options]  Stop using a local deployment for the
                                       current project, or globally disable
                                       local depoyments with --global
  mcp                                  Manage the Model Context Protocol server
                                       for Convex [BETA]
  help <command>                       Show help for given <command>
